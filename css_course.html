Cascading Style Sheets (CSS)

Duration: 4 hours

Objective

Style HTML elements using CSS to create visually appealing websites.

Topics Covered

  • Basic CSS Syntax and Selectors

    CSS uses selectors to target HTML elements. The basic syntax is as follows:

    
selector {
    property: value;
}
  • Box Model

    The box model includes margins, borders, padding, and the actual content. Understanding this helps in layout design.

    
.box {
    width: 300px;
    margin: 20px;
    padding: 10px;
    border: 2px solid black;
}
  • Positioning Elements

    CSS allows you to position elements in various ways: static, relative, absolute, fixed, and sticky.

  • Flexbox and CSS Grid

    These are modern layout techniques that provide flexibility and control over the design.

  • CSS Animations and Transitions

    Make your website interactive with CSS animations and transitions.

  • Responsive Design

    Use media queries to make your design responsive to different screen sizes.

  • Using CSS Frameworks

    Frameworks like Bootstrap and Tailwind help speed up the styling process.

Project

Build a responsive website using Flexbox and Grid.

Summary

This section introduced the basics of CSS, covering syntax, the box model, and modern layout techniques.

Exercises

  • Style an existing HTML page using CSS.
  • Experiment with Flexbox and Grid to create layouts.
