Objective

Learn how to structure a webpage using HTML.

Topics Covered

  • Basic Structure of an HTML Document

    Every HTML document has a basic structure consisting of doctype declaration, HTML, head, and body elements.

  • Common HTML Tags

    Learn about the most frequently used HTML tags:

    • Headings: <h1> to <h6>
    • Paragraphs: <p>
    • Links: <a href="URL">
    • Images: <img src="image.jpg" alt="description">
    • Lists: <ul> for unordered lists and <ol> for ordered lists.
    • Tables: Creating structured data with <table>, <tr>, <th>, and <td>.

  • Semantic HTML

    Semantic HTML improves accessibility and SEO. Use tags like <article>, <section>, <footer>, <header>, and <nav>.

  • Accessibility in HTML

    Ensure your website is usable by people with disabilities by using appropriate attributes and roles.

Project
