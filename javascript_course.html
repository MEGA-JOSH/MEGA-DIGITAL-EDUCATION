JavaScript (Basic)

Duration: 6 hours

Objective

Learn how to make websites dynamic using JavaScript.

Topics Covered

  • JavaScript Basics

    JavaScript is a programming language that allows you to create dynamic content on web pages. Key concepts include:

    • Variables: Store data values.
    • Data Types: Includes strings, numbers, booleans, etc.
    • Operators: Perform operations on variables and values.
    
let name = "John Doe";
let age = 30;
  • Functions, Loops, and Conditionals

    Functions are reusable blocks of code. Loops and conditionals allow you to execute code based on conditions.

    
function greet(name) {
    return "Hello, " + name + "!";
}

if (age >= 18) {
    console.log("Adult");
} else {
    console.log("Minor");
}
  • DOM Manipulation

    The Document Object Model (DOM) represents the structure of your web page. You can use JavaScript to manipulate HTML elements.

    
document.getElementById("myElement").innerText = "New Content!";
  • Event Handling

    JavaScript allows you to respond to user actions like clicks and form submissions.

    
document.getElementById("myButton").addEventListener("click", function() {
    alert("Button Clicked!");
});
  • Working with Arrays and Objects

    Arrays and objects are essential for storing collections of data.

    
let fruits = ["Apple", "Banana", "Cherry"];
let car = { brand: "Toyota", model: "Corolla" };
  • Introduction to ES6 Features

    ES6 introduced new syntax and features to JavaScript, including:

    • let and const: Block-scoped variables.
    • Arrow Functions: Shorter function syntax.
    • Template Literals: String interpolation.

Project

Create an interactive to-do list or calculator using JavaScript.

Summary

This section introduced the fundamentals of JavaScript, including variables, functions, DOM manipulation, and event handling.

Exercises

  • Build a simple JavaScript-based calculator.
  • Create a to-do list app that allows adding and removing items.
