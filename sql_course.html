Databases & SQL

Duration: 4 hours

Objective

Understand how to create and manage databases using SQL, and how to work with relational and NoSQL databases.

Topics Covered

1. Introduction to Databases (SQL vs. NoSQL)

A database is an organized collection of data. SQL databases, like MySQL, are structured with tables, columns, and rows. NoSQL databases, like MongoDB, store data in a more flexible format, often in JSON-like documents.

2. MySQL: Creating Databases and Tables

MySQL is a popular relational database management system (RDBMS). You can create a database and define its structure using SQL commands. Here is an example:


CREATE DATABASE my_blog;
USE my_blog;
CREATE TABLE users (
    id INT AUTO_INCREMENT PRIMARY KEY,
    username VARCHAR(50),
    email VARCHAR(100),
    password VARCHAR(255)
);

3. SQL Queries: SELECT, INSERT, UPDATE, DELETE

SQL (Structured Query Language) is used to interact with the database. Common queries include:

  • SELECT: Retrieve data from a table
  • INSERT: Add new data to a table
  • UPDATE: Modify existing data
  • DELETE: Remove data from a table

4. Relationships (One-to-Many, Many-to-Many)

In a relational database, tables can be linked through relationships. A one-to-many relationship connects a single record in one table to multiple records in another. A many-to-many relationship involves creating a linking table between two tables.
